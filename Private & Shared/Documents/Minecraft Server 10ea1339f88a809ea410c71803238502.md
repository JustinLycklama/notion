## 1. Auto-Start Minecraft Server on Boot (Windows)

Here‚Äôs how to make your Forge server start every time Windows boots:

1. Press **Win + R**, type `taskschd.msc`, press Enter.
2. In Task Scheduler:
    - **Create Task** (not Basic)
    - Under **General**:
        - Name it `Minecraft Server`
        - Check ‚ÄúRun with highest privileges‚Äù
    - Under **Triggers**:
        - Add ‚Üí Begin the task: **At log on**
    - Under **Actions**:
        - Add ‚Üí Action: **Start a program**
        - Browse to your `start_server.bat` file
    - Under **Conditions**:
        - Uncheck ‚ÄúStart only if on AC power‚Äù if you're running on a desktop

üí° You can also schedule it **"At startup"** instead of "At log on" if no one logs in.
